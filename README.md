## Description
This application is a responsive image gallery that pulls in images from UnSplash

## Live Preview
You can view the live preview of this project [here](https://tobyhorn.github.io/Image_Gallery/)

## How To
Simply load the webpage and hover over images to view details about that image. Clicking the Logo
